# OLSIF — Instruções para o Codex

Este projeto pertence ao Observatório de Logística Sustentável e Inovação Ferroviária (OLSIF).

O usuário é pesquisador, técnico-administrativo e gestor de projeto. Não é programador profissional.

Objetivo do projeto:
Criar um Dashboard Ferroviário OLSIF simples, auditável e de baixo custo para apoiar análises de logística ferroviária, intermodalidade, sustentabilidade, corredores bioceânicos e integração Brasil-Argentina.

Regras de trabalho:
- Explicar o plano antes de alterações grandes.
- Informar todos os arquivos criados ou alterados.
- Usar código simples, legível e comentado.
- Evitar dependências desnecessárias.
- Não inserir dados sensíveis, senhas, tokens ou informações institucionais privadas.
- Quando usar dados, indicar se são demonstrativos, estimados ou reais.
- Considerar que o usuário precisará entender e apresentar o projeto para públicos acadêmicos e institucionais.

Tecnologias preferidas:
- React
- Vite
- TypeScript
- Recharts futuramente para gráficos
- Leaflet futuramente para mapas
- Dados iniciais em JSON ou CSV local

Primeiro produto:
Dashboard Ferroviário OLSIF MVP.
